## Color Scheme

### Document Type Colors
- **Invoice**: Blue (#3B82F6)
- **Certificate**: Green (#10B981)
- **Contract**: Purple (#8B5CF6)
- **Spec Sheet**: Orange (#F59E0B)
- **Test Report**: Cyan (#06B6D4)
- **Other**: Gray (#6B7280)

### Expiry Status Colors
- **Expired**: Red (#EF4444)
- **Critical (< 7 days)**: Red (#EF4444)
- **Warning (< 30 days)**: Orange (#F59E0B)
- **Good**: Green (#10B981)

### UI Colors (from Vendora theme)
- **Primary**: Soft Royal Blue (hsl(224, 100%, 68%))
- **Accent**: Teal (hsl(173, 80%, 40%))
- **Background**: White / Dark Navy
- **Muted**: Light Gray / Dark Gray
- **Border**: Light Gray / Dark Gray

## Responsive Behavior

### Desktop (1200px+)
- Full sidebar visible
- 3-column grid view
- All features accessible

### Tablet (768px - 1199px)
- Collapsible sidebar
- 2-column grid view
- Compact toolbar

### Mobile (< 768px)
- Hidden sidebar (accessible via menu)
- 1-column grid view
- Stacked toolbar elements
- Full-width cards

## Interactive Elements

### Hover States
- Cards: Lift with shadow
- Buttons: Darken background
- Sidebar items: Highlight background
- Icons: Scale up slightly

### Active States
- Selected folder: Muted background
- Selected filter: Muted background
- Focused input: Ring outline

### Loading States
- Skeleton cards while loading
- Spinner in buttons during actions
- Progress indicators for uploads
